Compact monospace label for status, category, or metadata.

```jsx
<Badge tone="copper">New</Badge>
<Badge tone="success" dot>Live</Badge>
<Badge tone="lavender" variant="outline">AI-assisted</Badge>
```

Tones: `copper`, `lavender`, `neutral`, `success`, `warning`, `danger`. Variants: `soft` (tinted, default), `solid`, `outline`. Set `dot` for a status indicator. Always uppercase Space Mono.
