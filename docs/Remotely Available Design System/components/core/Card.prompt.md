Rounded surface container (14px radius) for grouping content on the dark page.

```jsx
<Card variant="glow" interactive>
  <Eyebrow>Result</Eyebrow>
  <h3>+318% qualified pipeline</h3>
</Card>
```

Variants: `default` (flat surface + soft shadow), `glow` (copper ring + glow, for highlighted/featured cards), `outline` (hairline only, lightweight). Set `interactive` to lift the card 3px on hover.
