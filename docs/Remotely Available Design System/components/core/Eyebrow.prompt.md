Monospace kicker label that sits above a serif title — the brand's signature `// section` motif.

```jsx
<Eyebrow>Services</Eyebrow>
<Eyebrow prefix="01 —" tone="muted">Discovery</Eyebrow>
```

Default prefix is `//`; pass `prefix=""` to drop it or a number like `01 —` for stepped lists. Tones: `copper` (default), `lavender`, `muted`. Always uppercase, 0.18em tracking.
