Outcome metric — the brand's signature way to present a result. Big serif number, mono label, optional delta.

```jsx
<Stat label="Qualified pipeline" value="+318%" delta="in 90 days" />
<Stat label="Hours reclaimed / wk" value="42" align="center" />
```

`value` is the headline figure (serif). `label` renders as a mono eyebrow above it. `delta` is a small mono context line below — tone it `success` (default), `danger`, or `lavender`.
