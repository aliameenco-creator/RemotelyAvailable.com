Text field on the dark palette — inset well, mono label, copper focus ring.

```jsx
<Input label="Work email" placeholder="you@company.com" hint="We reply within a day." />
<Input label="Budget" error="Required" />
```

`label` renders as a mono uppercase label. `error` overrides `hint` and turns the ring red. Use `leading`/`trailing` for icons or adornments. Focus shows a copper glow ring.
